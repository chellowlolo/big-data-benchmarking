Big Data Benchmarking

This dashboard summarize the results from the Big Data Benchmarking project. These charts are generated directly from the CSV file. The charts are explained as follows:

  • A DataTable displays the raw output from the CSV file.
  • A BarChart displaying the average query execution time for each database based on the concurrency factor.
  • A ScatterChart displays a point for every query comparing the execution time vs the number of rows returned from the query.
  • A BarChart displaying the average query execution time for each database based on the query category.
  • A BarChart displaying the average query execution time for each database based on the table size.
  • A BarChart displaying the average query execution time for each query based on the database.
  • A LineChart displaying the average query execution time for each query based on the database.
  • A DataTable summarizing the average query execution times per database.

NOTE: Use the widgets to control the data in the DataTable. Then click the "Download" button to export the raw data to a CSV file.
